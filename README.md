# Portfolio - Salman Abdurrahman

Portfolio website sederhana yang menampilkan profil, keahlian, dan proyek-proyek web development.

## Tech Stack

- **HTML5** - Struktur halaman web
- **CSS3** - Styling dengan pendekatan Brutalism Design
- **JavaScript** - Interaktivitas dan mobile menu
- **Boxicons** - Icon library
- **Google Fonts** - Typography (Inter)

## Struktur Projek

```
belajar-dasar-pemrograman-web/
├── index.html         # Halaman utama
├── css/
│   └── style.css      # Styling dan animasi
├── js/
│   └── script.js      # Logic untuk mobile menu
├── assets/            # Folder untuk gambar dan aset lainnya
└── README.md          # Dokumentasi projek
```

## Fitur

- **Responsive Design** - Tampilan optimal di berbagai ukuran layar
- **Mobile Navigation** - Hamburger menu untuk perangkat mobile
- **Smooth Scrolling** - Navigasi halus antar section
- **Modern UI** - Design dengan pendekatan Brutalism style
- **Interactive Elements** - Hover effects dan animasi

## Cara Menjalankan

1. Clone repository ini:
```bash
git clone https://github.com/salmanabdurrahman/belajar-dasar-pemrograman-web.git
```

2. Masuk ke direktori projek:
```bash
cd belajar-dasar-pemrograman-web
```

3. Buka file `index.html` di browser favorit Anda, atau gunakan live server di VS Code.
